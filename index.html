Our Audit Process Flow

We have designed a comprehensive chain agnostic methodology review process to detect logical errors, security susceptabilities, and enhance opportunities within code by verifying economic logic to ensure the security of smart contracts and highlighting privileged functionality to create a report that is easy to understand for developers and community members alike.

  • 1

    Automated Testing

    Simulation of a variety of interactions with your Smart Contract on a test blockchain leveraging a combination of automated test tools and manual testing to determine if any security vulnerabilities exist.

  • 2

    In-Depth Manual Review

    Manual line-by-line code reviews to ensure the logic behind each function is sound and safe from common and economic attack vectors.

  • 3

    Resolution of Issues

    Consulting with the team to provide our recommendations to ensure the code's security and optimize its gas efficiency, if possible.

  • 4

    Publishing the Audit Report

    Boiling down results and findings into an easy to read report tailored to the project that highlights resolved issues and any risks that exist to the project or its users, along with any remaining suggested remediation measures.
